Jogos Digitais

  • Graduação: Desenvolvimento de Jogos Digitais na Unicuritiba (2018-2021)

  • Design de Jogos: Estudei todo o processo de criação e desenvolvimento, documentando tudo no GDD, como ideias, mecânicas de jogo, prototipagem, balanceamento e testes.
  • Programação de jogos: Aprendi habilidades de programação para criar jogos, também utilizando engines como a Unity e a Unreal, aplicando lógica de jogabilidade e IA.
  • Arte de jogos: Aprendi técnicas de design visual para jogos, incluindo modelagem 3D, animação, concept art, texturização e iluminação.
  • Narrativas: Construi enredos, personagens e diálogos para jogos, usando técnicas como a Jornada do heroi e mais.
  • UX e UI: Aprendi técnicas de interface e experiência do usuário para jogos digitais, garantindo experiências de jogo faceis e agradáveis.
  • Estudos de jogos: Estudo acadêmico e análise crítica de jogos digitais, incluindo a história, cultura, estético, narrativa e impacto social.
  • Jogos Educativos: Estudei técnicas de criação de jogos digitais para fins educacionais, a fim de auxiliar no processo de aprendizado e retenção de conhecimento. Muitas vezes utilizando as ODS da ONU como ideia base.
  • Graduação: Desenvolvimento de Jogos Digitais na Unicuritiba (2018-2021).

DEV WEB

  • Graduação: Desenvolvimento Web na plataforma Udemy

  • Tríade Front-end: Aprendi as 3 tecnologias mais importantes para a criaçao de aplicações web, o HTML5, CSS3 e o JavaScript.
  • Front-end avançado: Após aprender a tríade me aprofundei em mais fremeworks, para o desenvolvimento web, como o SASS, Bootstrap, ES6 e JQuery
  • Back-end: Comecei aprendendo o servidor Apache, PHP e MySQL, me aprofundei em PHP7, OO, arquitetura MVC, API com Slim Framework e mobile com IONIC.
